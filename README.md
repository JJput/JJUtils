常用公共工具类，结合个人使用以及复制部份[AnndroidUtilCode](https://github.com/Blankj/AndroidUtilCode)的工具类，感谢这些作者，写的非常好，有兴趣的同学也可以多看看。

## 引用
在项目build.gradle下添加：

    allprojects {
    	repositories {
    		...
    		maven { url 'https://jitpack.io' }
    	}
    }
app -> build.gradle

    dependencies {
            implementation 'com.github.JJput:MVPLibrary:v1.0'
    }
